Documentación estilos Macondo

Esta documentación tiene como objetivo explicar todas las clases y la funcionalidad que se crearon en el montaje de Macondo. Incluye librería de colores, las clases para los bloques y tipografías.

Datos técnicos

El proyecto se realizó en Stylus CSS y el archivo a editar es user.styl. El css está apoyado por el Framework Boostrap.

En las primeras líneas de la hoja de estilos de stylus, está la parametrización del portal, esto incluye:

  • Tipo de fuentes
  • Color principal y secundario del portal
  • Librería de colores*
  • Color de los enlaces
  • Personalización de botones y botón default

* Los colores se declaran como variables y solo son útiles dentro de la hoja de estilos de stylus. Sin embargo, sí desean que estos colores se hagan disponibles para las diferentes etiquetas como colores de fondo, colores de fuente, enlaces, entre otros; se debe agregar en el array colors.

Tener en cuenta

Para generar las etiquetas para colores adicionales deben incluirse en el array "colors" en la hoja de stylus y compilarse.

El color primary, es el color principal del portal y es el color de la entidad.

No Background

Padding

Los bloques en Nexura 7 tienen padding por defecto. Esta etiqueta permite editar el padding. Las clases son:

  • No padding: no-padding
  • No padding superior e inferior: no-padding-top-bottom
  • No padding laterales: menu-float-left-right
  • Padding superior e inferior 1em (16px): padding-top-bottom-1x
  • Padding laterales 1em(16px): padding-left-right-1x

Texto a 3 columnas

Convierte un texto a 3 columnas. Cada sección en el portal toma como base esta clase y personaliza sus colores o presentaciones. La clases es content-3-col. Si desea que solo sea visible para tablets y desktop, la clase es content-3-col-md. El ejemplo es en el footer

Personalización de bloques

Animaciones a Imágenes Bloque Dinámico

Permite hacer diferentes animaciones a las imágenes en el estado hover. Las animaciones son:

  • Efecto de escalar y girar overlay-spin
  • Efecto de escalar overlay-scale-img
  • Efecto en los bordes: overlay-border-color

Bloque de iconos Bloque Dinámico

Personaliza el bloque dinámico y mejora la visualización de los iconos con sus respectivos titulos. Cada sección en el portal toma como base esta clase y personaliza sus colores o presentaciones. La clase es block-icon. El ejemplo es el bloque de iconos en la sección tramites en macondo

Contenido Editor Bloque Editor de Texto

Estandariza en todo el portal el contenido del editor. Cada sección en el portal toma como base esta clase y personaliza sus colores o presentaciones. La clase es content-block. El ejemplo es la sección turismo en macondo o en el footer

Grupo de botones inline y list Bloque Dinámico

Grupo de botones en posicion horizontal (inline) y vertical (lista). Cada sección en el portal toma como base esta clase y personaliza sus colores o presentaciones. Las clases son btn-group-inline y btn-group-list. El ejemplo es el bloque de iconos en la sección turismo en macondo

Listado de enlaces Bloque Enlaces Generales

Ajusta el listado de enlaces de forma horizontal (inlines) o vertical del bloque enlaces generales. La clase es list-vertical posición vertical y list-inline posición horizontal. Sí desea eliminar viñeta puede agregar list-unstyled

El ejemplo se puede ver en la sección de planes y resultados

Menú laterales fijos Bloque Dinámico

Son menús de iconos laterales. La clase para volver un bloque dinámico en menu lateral es menu-float-sidebar. Si desea que sea fijo es menu-float-fixed. Ver solo el icono menu-only-icon. Ver ejemplo en Macondo, sección de feedback.

Para posicionar el bloque es asi:

  • Arriba: menu-float-top
  • Abajo: menu-float-bottom
  • Centrado vertical: menu-float-middle
  • Centrado derecha: menu-float-right
  • Centrado izquierda: menu-float-left

Padding Cualquier bloque

Los bloques en Nexura 7 tienen padding por defecto. Esta etiqueta permite editar el padding. Las clases son:

  • No padding: no-padding
  • No padding superior e inferior: no-padding-top-bottom
  • No padding laterales: menu-float-left-right
  • Padding superior e inferior 1em (16px): padding-top-bottom-1x
  • Padding laterales 1em(16px): padding-left-right-1x

Solo iconos Bloque Dinámico

Personaliza este bloque y elimina la etiqueta del título. La clase es only-icon

Tabs Bloque Tabs

Personaliza este bloque, simplifica el diseño. Cada sección donde se llama esta clase, la personaliza en cuanto a colores. La clase es tab-general

Título Zonas Bloques Zonas

Esta clase modifica las etiquetas H2 y H3 de los títulos de cada una de las secciones, se debe aplicar al bloque. Cada zona toma como referencia esta clase y aplica sus propios colores a esta clase. La clase es title-zone
